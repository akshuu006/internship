Difference between Inline and Block Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples: <span>, <a>.
Block elements start on a new line and take up the full width available by default. Examples: <div>, <p>.
Inline-block elements behave like inline elements in flow (they sit side by side) but accept width, height, padding, and margins like block elements.
